战友融创（北京）科技有限公司，秉承战友集团十五年“服务国防，服务社会”企业宗旨，积极适应国家实施军民融合发展战略，坚守忠诚可靠、科技强军价值追求，广聚军队自主择业技术研发和科技管理英才，发扬军人执着担当、严准细实过硬作风，瞄准解决部队战斗力生成提高技术短板，努力打造军事科技领域军民融合的时代铁军。

公司业务范围主要包括：基础材料技术、装备信息技术、特种机器人技术和主战装备配套装置的研发集成和成果转化。目前，在高分子防爆材料、高性能润滑油脂、装备密封技术、新型减震材料、纸片电池、KH水下连接器、控制线束、低温高频启动电源、机动平台北斗智能信息终端、军事物流智能化系统、单兵生理信息监测系统、核辐射污染监测预警系统、高精度火灾抑爆系统、格斗无人机、油电混合动力无人机平台、红外侦测无人机等方面，具有成熟的产品技术储备和市场储备。更多>
